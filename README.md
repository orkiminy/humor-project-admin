# Humor Project Admin

An admin dashboard for managing a humor/comedy content platform. Provides CRUD interfaces for captions, humor flavors, LLM models, providers, prompt chains, users, images, and related data.

##  Tech Stack
- **Framework:** Next.js 16 (App Router) + React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Backend:** Supabase (SSR + JS client)
- **Charts:** Recharts
- **Testing:** Vitest + Playwright

##  Features
- Full CRUD flows for content, users, and LLM configuration
- Analytics dashboards powered by Recharts
- Server-side rendering with Supabase SSR
- Unit and end-to-end test coverage
